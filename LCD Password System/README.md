# 🔐 Smart Password Lock System (Arduino)

## 📌 Description
A simple security system using Arduino, Keypad, and LCD display.
Users must enter a password to gain access.

## ⚙️ Features
- 4-digit password authentication
- Hidden input (shown as * )
- 3 attempts limit
- System lock after failure
- Success message with custom heart display

## 🧩 Components
- Arduino UNO
- 4x4 Keypad
- 16x2 LCD

## 💡 How it works
- User enters password '1234'
- Press # to confirm
- '*' clears input
- If correct → Welcome screen
- If wrong → attempts decrease
