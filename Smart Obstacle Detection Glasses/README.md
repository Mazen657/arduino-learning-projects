# 🕶 Smart Obstacle Detection Glasses

## 📌 Description
A simple security system using Arduino, Keypad, and LCD display.
Users must enter a password to gain access.

## ⚙️ Features
- Multi-direction obstacle detection (Left, Front, Right)
- Real-time distance measurement using ultrasonic sensors
- Direction-based alert system
- Adjustable warning thresholds

## 🧩 Components
- 3 Buzzer for alert system (Left, Front, Right)
- 3 Ultrasonic Sensors (Left, Front, Right)
- Arduino UNO

## 💡 How it works
- Each sensor continuously measures distance
- If an obstacle is detected:
  - Left sensor → alert left side
  - Front sensor → immediate alert
  - Right sensor → alert right side
- The closer the object, the faster the buzzer frequency
