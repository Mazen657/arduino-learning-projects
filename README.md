# ⚡ Arduino Learning Projects

Welcome to my Arduino learning repository! 🚀  
This repo contains a collection of small embedded systems projects I built while learning Arduino and hardware programming.

Each project focuses on understanding a specific concept such as sensors, input/output systems, displays, and basic embedded logic.

---

## 📌 About This Repository

This is a **learning-focused repository**, not production-level projects.  
The goal is to practice and improve skills in:

- Embedded systems basics
- Arduino programming (C/C++)
- Working with sensors and modules
- Circuit design using Tinkercad
- Problem solving using hardware logic

---

## 🧠 What You Will Find Here

Each project folder typically includes:

- 💻 Source code (`.ino` files)
- 🔌 Circuit simulation (Tinkercad screenshot)
- 📖 Short explanation of how it works
- 🧩 Components used

## 📂 Projects List

**Concepts used:**
- Keypad input handling
- LCD display control
- Basic authentication logic
